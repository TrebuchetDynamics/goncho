---
title: Negative Memory
description: Failed paths and contradictions are part of intelligent memory.
---

Useful memory is not only a list of successful facts.

Agents also need to remember:

- failed approaches;
- rejected strategies;
- contradictions;
- unsafe or expensive paths;
- stale assumptions;
- tool behaviors that looked promising but did not work.

This is negative memory: remembered caution.

:::note[Current status]
Current Goncho can store and retrieve failure notes through existing [memory tools](/reference/memory-tools/) and [`Conclude`/`Search`](/reference/core-api/) API surfaces. First-class negative-memory review is architecture direction.
:::
