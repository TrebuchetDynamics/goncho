---
title: Glossary
description: Canonical terms used in Goncho docs.
---

For deeper context, see [Evidence, Claims, and Beliefs](/concepts/evidence-claims-beliefs/), [Orientation Packs](/concepts/orientation-packs/), and [Negative Memory](/concepts/negative-memory/).

| Term | Meaning |
| --- | --- |
| Evidence | Raw observations, transcripts, tool outputs, or events preserved before interpretation. |
| Claim | A distilled statement derived from evidence. |
| Belief | A claim accepted for a scope and time, possibly with qualitative confidence. |
| Scope | The boundary where memory applies: workspace, peer, observer, project, session, or task. |
| Orientation | The compact context product returned to the agent for action. |
| Negative memory | Remembered failure, contradiction, rejected path, or caution. |
| Review | Inspection and correction of memory when evidence conflicts, ages, or proves too broad. |
| Repair | Human or agent correction of stale, false, conflicting, or overbroad memory. |
| Consolidation | The process of turning session evidence into more durable memory products. |
| Peer | The person, agent, or entity the memory is about. |
| Observer | The perspective from which a peer or event is represented. |
| Workspace | A memory isolation boundary for an agent runtime or project. |
