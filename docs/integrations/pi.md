# Pi Connector

Status: supported-plan

Pi support is local-first extension/settings planning. Use `goncho connect pi --plan --config ~/.pi/agent/settings.json --extension ~/.pi/agent/extensions/goncho --addr 127.0.0.1:8765`.

The preview lists Pi settings JSON, extension file paths, generated host hook events, and the local `goncho-server` URL. It does not copy files or register extensions automatically.
